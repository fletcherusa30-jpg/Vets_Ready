# scripts/

This folder contains automation, build, deployment, and validation scripts for the project.

## Usage Guidelines
- Store PowerShell, Bash, Node, Python, and other automation scripts here.
- Each script should have a header comment with usage instructions, author, and date.
- Document all scripts in this README for discoverability.
- Use clear, descriptive names for all scripts.

## Example Contents
- `build.sh`, `deploy.ps1`, `validate-structure.ps1`, `seed-database.ps1`

---

**Policy:** All scripts must be documented, reviewed, and tested before use in production.
