# SQL/

This folder contains all database schema files, migration scripts, and seed data for the project.

## Usage Guidelines
- Store only version-controlled schema, migration, and seed files.
- Do not store generated database dumps or backups here.
- Use clear naming conventions for migration and seed files.

## Example Contents
- `schema.sql`, `init_db.sql`, `migrations/`, `seed_data.sql`

---

**Policy:** All files must be reviewed and tested before applying to production databases.
